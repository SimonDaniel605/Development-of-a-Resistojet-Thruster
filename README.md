# Development-of-a-Resistojet-Thruster
This repository contains the design, modelling, simulation, and development work for a low-power electrothermal resistojet propulsion system intended for nanosatellite station-keeping and applications.

The project forms part of an MEng (Electronic Engineering) research project at Stellenbosch University and focuses on developing a compact, efficient, and manufacturable propulsion system capable of providing precise orbital manoeuvring for future small satellite missions.

The work includes analytical modelling, CFD validation, hardware development, electronics, embedded software, and experimental test infrastructure.
